Q: What is ph? What is mu?

A: ph (Prediction Horizon) represents how far into the future we want to predict in terms of time. ph defines the forecasting window. mu likely acts as a discount factor or exponential smoothing parameter in time series modeling.

If mu is used in an Exponential Moving Average (EMA), it controls how much past data influences the present. If used in reinforcement learning or dynamic models, mu determines how much weight is given to previous observations. mu is a weighting factor (0 to 1) that controls the influence of past values on predictions.

# At every iteration of the for loop a new data sample is acquired
for i in range(2, n_s+1):# start out with 2 leading datapoints
    #get x and y data "available" for our prediction
    ts_tmp = ts[:i]
    ys_tmp = ys[:i]
    ns = len(ys_tmp)
    
    
    weights = np.ones(ns)*mu
    for k in range(ns):
        #adjust weights to be downweighted according to their timestep away from our prediction
        weights[k] = mu ** (ns - k)
    weights = np.flip(weights, 0)
        
    #perform linear regression on "available" data using the mu-adjusted weights
    lm_tmp = LinearRegression() 
    model_tmp = lm_tmp.fit(ts_tmp, ys_tmp, sample_weight=weights)
    
    #store model coefficients and intercepts to compute prediction
    m_tmp = model_tmp.coef_[0][0] # Slope
    q_tmp = model_tmp.intercept_[0]  # Intercept

    #use ph to make the model prediction according to the prediction time
    tp = ts.iloc[i-1, 0] + ph  # Future time prediction
    yp = m_tmp * tp + q_tmp  # Future GAP prediction
      
    tp_pred[i-2] = tp    
    yp_pred[i-2] = yp
